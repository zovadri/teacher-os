- نسخة تجريبية متكاملة لمدرس واحد أو مركز تعليمي صغير

- جاهز للعرض على العملاء والمستثمرين

- Node.js 18+
- pnpm (أو npm/yarn)
- متصفح حديث

## البدء السريع

```bash
npx pnpm install
```
